# Conditionals

The `if` statement is used to perform different actions based on different test conditions. 

## Usage

```xs
if (x == 42) {
    "The answer to everything.";
} else {
    "Just a number.";
};
```

```xs
var result = if (x == 42) {
    "The answer to everything.";
} else {
    "Just a number.";
};
result;
```
